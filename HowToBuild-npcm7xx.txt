
Requirements:
=============
- linux ARM cross compiler
- git installed

Building:
=========
- Choose a directory
- run:
	git clone https://github.com/Nuvoton-Israel/u-boot.git
	cd u-boot
	make PolegSVB_config CROSS_COMPILE=arm-none-linux-gnueabi- SUBLEVEL=6.9
	make all CROSS_COMPILE=arm-none-linux-gnueabi- SUBLEVEL=6.9
